## About
A focused developer workspace combining daily Kanban task management with
image polygon annotation — in one tool, without context switching.

## Core Surfaces
- `/login` — Email + password authentication
- `/signup` — New account registration
- `/tasks` — Three-column Kanban board (To Do, In Progress, Done) with date
  navigation, priority levels, tags, and drag-and-drop ordering
- `/annotate` — Image upload, polygon annotation drawing, polygon management

## Identity
"404 Project Not Found" — a deliberate developer-friendly name. The visual
system uses the 404 motif as a restrained title-block element only — not as
repeated error copy throughout the application.

## Design Direction
**Drafting Table** — approved 2026-07-13.

Warm drafting-paper surfaces, deep blueprint-indigo typography, blueprint-cyan
primary actions, semantic color states (coral/amber/green/blue), Space Grotesk
paired with JetBrains Mono. Subtle grid pattern applied only to workspace
backgrounds (board column bodies, annotation canvas stage). Technical details
used only where they improve hierarchy or communicate state. Copy is human and
understandable throughout.
